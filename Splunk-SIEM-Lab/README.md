# Splunk SIEM Lab

## Objective

Demonstrate SIEM-based security monitoring and event analysis using Splunk Enterprise within a virtualized lab environment.

## Tools Used

* Splunk Enterprise
* Windows Server
* Hyper-V
* Windows Event Logs

## Skills Demonstrated

* SIEM Monitoring
* Log Collection and Analysis
* Security Event Investigation
* Failed Login Detection
* Event Correlation
* Security Monitoring
* Incident Detection
* Security Reporting

## Outcome

Successfully configured Splunk Enterprise to collect Windows event logs, detect failed login attempts using EventCode 4625, and analyze suspicious activities through dashboards, searches, and visualizations.
