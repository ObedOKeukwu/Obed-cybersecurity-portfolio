# Traffic Capture and Analysis Lab

## Objective

Demonstrate VLAN segmentation, inter-VLAN routing, SPAN traffic mirroring, and packet analysis using Wireshark within a controlled Cisco Modeling Labs (CML) environment.

## Tools Used

* Wireshark
* FortiGate Firewall
* Cisco Modeling Labs (CML)
* Kali Linux
* Layer 2 Switch
* Windows Client VMs

## Skills Demonstrated

* VLAN Configuration
* SPAN Port Configuration
* Inter-VLAN Routing
* Firewall Policy Configuration
* Packet Capture and Analysis
* ARP Analysis
* ICMP Analysis
* DNS Traffic Analysis
* TCP Handshake Analysis
* Network Troubleshooting

## Outcome

Successfully configured VLAN segmentation and SPAN traffic mirroring to analyze ARP, ICMP, DNS, and TCP traffic using Wireshark while validating firewall-controlled inter-VLAN communication.
