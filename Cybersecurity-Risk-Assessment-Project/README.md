# Cybersecurity Risk Assessment Project

This project presents a cybersecurity risk assessment conducted on a Linux-based Financial System environment.

The assessment includes:
- Risk analysis
- Attack tree analysis
- SQL Injection assessment
- XSS and Buffer Overflow risks
- Security control recommendations
- MFA implementation strategy
- IDS/IPS recommendations
- NIST SP 800-30 aligned methodology

Technologies and frameworks referenced:
- Microsoft Entra ID
- Wazuh
- Suricata
- Palo Alto NGFW
- OWASP Top 10
- NIST SP 800-30
