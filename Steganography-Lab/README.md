
# Steganography Lab

## Objective

Demonstrate the use of steganography techniques to hide and extract information within digital media files in a controlled lab environment.

## Tools Used

* OpenStego / Steghide / SilentEye
* Windows Environment
* Image Files
* Virtual Lab Environment

## Skills Demonstrated

* Steganography Concepts
* Data Hiding Techniques
* File Embedding and Extraction
* Information Security Awareness
* Secure Data Handling

## Outcome

Successfully embedded and extracted hidden information within digital media files while understanding the security implications and detection considerations associated with steganography techniques.
