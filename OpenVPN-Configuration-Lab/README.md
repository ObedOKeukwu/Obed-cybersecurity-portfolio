# OpenVPN Configuration Lab

## Objective

Demonstrate the configuration and validation of a secure VPN connection using OpenVPN within a controlled virtual lab environment.

## Tools Used

* OpenVPN
* Windows Client
* Virtual Machines
* Hyper-V / VMware
* Wireshark (Optional)

## Skills Demonstrated

* VPN Configuration
* Secure Remote Connectivity
* Encrypted Communications
* Network Security
* Client-Server Configuration
* Connectivity Troubleshooting

## Outcome

Successfully configured a secure VPN tunnel between systems in a virtualized environment and verified encrypted communication and remote connectivity functionality.
