# SSL/TLS Inspection Lab

## Objective

Demonstrated SSL/TLS inspection within a controlled virtual environment by configuring firewall-based HTTPS inspection, installing trusted certificates, and analyzing encrypted traffic using Wireshark.

## Tools Used

* FortiGate Firewall
* Wireshark
* Windows Client VM
* VMware / CML
* HTTPS Web Traffic

## Skills Demonstrated

* SSL/TLS Inspection
* HTTPS Traffic Analysis
* Certificate Management
* Firewall Policy Configuration
* Encrypted Traffic Decryption
* Wireshark Packet Analysis
* Application Layer Security
* Security Compliance Awareness

## Outcome

Successfully configured SSL/TLS interception using a virtual firewall, validated certificate trust behavior on client systems, and analyzed encrypted/decrypted HTTPS traffic while understanding associated privacy and compliance considerations.
