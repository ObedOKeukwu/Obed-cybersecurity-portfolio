# Healthcare Security Policy Framework Proposal

This project presents a cybersecurity policy framework proposal developed for a hospital environment.

The presentation focuses on:
- Patient data protection
- Healthcare cybersecurity
- Incident response
- Employee security awareness
- KPI-driven security management
- HIPAA/PIPEDA compliance alignment
- Security governance and policy rollout

Topics covered:
- Access control
- Network security
- Data protection
- Security awareness training
- KPI measurement and reporting
- Business alignment with cybersecurity goals
